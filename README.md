### 설치 방법
#### 기본 설정
먼저 아래 명령 실행:
```pwsh
winget install --id OpenJS.NodeJS.LTS Python.Python.3
node -v
npm -v
```

그다음 터미널 껏다 켠 후:

```pwsh
npm install
npm run build
npm start
```

이제 localhost:5000 에 들어가세요.
